# LastAirBnb

[Heroku link][heroku]

[heroku]: http://lastairbnb.herokuapp.com

## Minimum Viable Product
LastAirBnb is an Avatar themed AirBnb clone built with Rails and Backbone. Users
are able to:

- [ ] Create accounts
- [ ] Log in/out
- [ ] View rooms
- [ ] List rooms
- [ ] Review rooms
- [ ] Book rooms
- [ ] Search for rooms
